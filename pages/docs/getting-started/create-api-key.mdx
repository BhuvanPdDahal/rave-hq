# Create API Key

Now let's focus on how you can create an API key for your app. Follow the following steps to do it:

* Click the `Create a new app` button in the dashboard page and create a new app in your account.
* After creating the app, navigate to the apps page and click on the app to go to the app details page.
* In the app details, you'll find a button that says `Create API Key` if you haven't already created an API key, simply click it.
* After that, a modal with your API key will popup, store the key securely and close the modal.

That's it, you now have the API key for your app.
