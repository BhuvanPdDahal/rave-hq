# Introduction

Welcome to the official documentation for RaveHQ! This comprehensive guide is designed to equip you with everything you need to get started using our platform and leverage its full potential.

## What is RaveHQ?

RaveHQ is a testimonial collection and management platform. We empower you to collect and manage postive feedback from your customers that you can showcase to boost your conversion rates and build legitimacy for your business through easy API integration.

## Who Should Use This Documentation?

This documentation caters to a wide range of users, including:

* **New Users**: If you're just starting out with RaveHQ, this guide will walk you through the initial setup process and introduce you to the core functionalities.
* **Existing Users**: Even if you're already familiar with the basics, this documentation serves as a valuable reference for exploring advanced features, troubleshooting issues, and staying updated with the latest functionalities.
* **Developers**: If you're integrating RaveHQ with your own applications, this guide provides detailed information about our APIs and how to leverage them effectively.
